## How to use

```
# Install canvas-sketch CLI tool globally
npm install canvas-sketch-cli -g
clone project
cd <project-directory>
yarn && yarn start
```

While in the browser, hit `Cmd + Shift + S` or `Ctrl + Shift + S` to export a high-resolution GIF
